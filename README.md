# Afzal Pro — Urdu Typing Master Pro

A professional, offline-capable **Urdu Phonetic Typing Practice** web application built for PPSC Junior Clerk exam preparation and general Urdu typing mastery.

🌐 **Live App:** [afzalprourdu.netlify.app](https://afzalprourdu.netlify.app)

---

## Features

- ✅ Complete Urdu Phonetic Unicode Keyboard Layout (PDF-accurate)
- ✅ 20 structured lessons — Home Row → Top Row → Number Row → Bottom Row
- ✅ Shift-layer lessons for all rows
- ✅ PPSC exam-style paragraph practice
- ✅ Live WPM, Accuracy, and Character tracking
- ✅ Interactive keyboard visualizer with key highlighting
- ✅ QWERTY mode + Urdu Phonetic keyboard mode
- ✅ Timed typing test (30s / 60s / 120s)
- ✅ Progress tracking with session history
- ✅ Certificate of achievement with PDF/print export
- ✅ Dark / Light / System themes
- ✅ Fully responsive — works on mobile and desktop
- ✅ 100% offline — no internet required after loading

---

## Keyboard Layout

Based on the official **Urdu Phonetic Unicode Keyboard Layout** (NLA Standard).

| Mode | Description |
|------|-------------|
| QWERTY Mode | Keep Windows keyboard in English. App maps keys to Urdu internally. |
| Urdu Phonetic Mode | Switch Windows keyboard to Urdu Phonetic. Direct input accepted. |

---

## Lesson Structure

| Group | Lessons |
|-------|---------|
| 🏠 Home Row | Singles → Words → Paragraphs → Shift → Mixed |
| ⬆ Top Row | Singles → Words → Paragraphs → Shift → Mixed |
| 🔢 Number Row | Singles → Numbers + Diacritics |
| ⬇ Bottom Row | Singles → Words → PPSC Paragraphs → Shift → Speed → Exam |

---

## Usage

1. Open `index.html` in any modern browser (Chrome recommended)
2. Select your keyboard mode (QWERTY recommended)
3. Choose a lesson group and phase
4. Start typing — the keyboard visualizer guides every key
5. Complete lessons to earn certificates

---

## Deployment

This is a **zero-dependency single-file app**.  
No build step. No npm. No server required.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/afzal-pro-urdu-typing.git

# Open directly in browser
open index.html
```
